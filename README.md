# pbi

Case realizado para uma entrevista técnica.

Solicitado:
Avaliar o desempenho das vendas mensais e anuais 
Comparar as vendas entre diferentes clientes e setores 
Com o objetivo de: Definir as prioridades e a estratégia comercial para o próximo mês 
Desenvolver ações e estratégias específicas para melhorar meus resultados

Objetivo do case (traduzindo para BI)

1.	Como as vendas evoluem no tempo? (mensal e anual)
2.	Quem vende mais e menos? (clientes)
3.	Quais setores performam melhor?
4.	Onde focar esforços no próximo mês?

Esse dashboard foi construído para apoiar decisões estratégicas e táticas, separando uma visão executiva de performance geral e uma visão analítica focada em eficiência de produto. Modelei os dados em star schema, utilizei KPIs financeiros e operacionais e foquei em clareza visual e perguntas reais de negócio.
